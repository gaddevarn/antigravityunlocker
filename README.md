# Antigravity в России без VPN и смены региона аккаунта Google.

Программа для windows, которая позволяет аккаунту Google любой страны (в том числе России) использовать Antigravity.

Видеоинструкция: https://youtu.be/mQvnQkua734

Процедура разблокировки проводится один раз для компьютера. После этого можно авторизовываться с любыми аккаунтами Google из любых стран.


# Где скачать?

Нажимаете на [Releases](https://github.com/confeden/Antigravity/releases) и скачиваете последнюю версию.

Для разблокировки нужно ввести лицензионный код, взять можно из группы, бесплатно (в закреплённых сообщениях) [t.me/nova_txt/69864](https://t.me/nova_txt/69864)

# Как разблокировать Antigravity?
1) скачиваете Antigravity IDE с официального сайта https://antigravity.google/releases и устанавливаете (пока не запускайте)
2) применяете анлокер, после этого его можно закрыть
3) запускаете Antigravity IDE и входите в аккаунт Google
4) можно использовать Antigravity CLI и Antigravity Desktop

# Работает ли анлокер для 2+ версии?
Да, он работает для всех агентских программ для кодинга от Google:
- Antigravity
- Antigravity IDE
- Antigravity CLI
- Gemini CLI nightly (только при наличии платных подписок Гугла)

# Чем этот анлокер отличается от другого на Гитхабе?

Тем, что мой анлокер (оригинал) взламывался с помощью нейросети другим человеком, и теперь он выдаёт его как за свою собственную разработку ([подделку](https://github.com/AvenCores/open-antigravity-patcher)). Что там есть помимо моего алгоритма и насколько безопасно им пользоваться - известно только ему.
