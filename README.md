# Antigravity
Antigravity в России без VPN и смены региона аккаунта Google.

Программа, которая позволяет аккаунту Google любой страны (в том числе России) использовать Antigravity без VPN.
#
Процедура разблокировки проводится один раз для компьютера. Если при дальнейшем обновлении Antigravity способ разблокировки перестанет работать, я выпущу обновление.

Версия актуальна на 8 февраля 2026.
Для разблокировки нужно ввести лицензионный код, который можно запросить в Телеграм чате:

[t.me/nova_txt](https://t.me/nova_txt)
